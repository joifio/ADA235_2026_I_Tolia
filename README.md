# ADA235_2026

Αυτό το repository περιέχει την εργασία σύνθεσης σε SuperCollider με χρήση εργαλείου Τεχνητής Νοημοσύνης.

## Περιεχόμενα

- `Φάκελος κώδικα/`  
  Περιέχει το αρχείο SuperCollider με τον κώδικα της σύνθεσης, και οδηγίες χρήσεις του.

- `Φάκελος τεκμηρίωσης διαδικασίας (query log)/`  
  Περιέχει το αρχείο με τα prompts / queries και τη διαδικασία αλληλεπίδρασης με το εργαλείο ΤΝ.

- `ΠεριγραφήΕργασίας.md`  
  Περιέχει την περιγραφή και το θεωρητικό κομμάτι του έργου.

## Έργο

**Soft Corruption**  
Generative ambient / glitch ηχοτοπίο στο SuperCollider.
